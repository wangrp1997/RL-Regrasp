# Deep learning method

Usage
1. Data Collection
   
Comment out `evaluate()` in the main() function of env.py

Uncomment `env_test(**kwargs)`, then run 
`python env.py` 

2. Train

`cd model_train`
then run:
`python main-LSTM.py`

3. Evaluate

`cd ..`
Comment out env_test(**kwargs) in the main() function of env.py
Uncomment `evaluate()`, then run
`python env.py`
